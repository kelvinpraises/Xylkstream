# Xylkstream

Multi-tenant vesting platform where your unvested funds work for you. Stream payments to anyone while AI agents automatically deploy idle capital to the highest-yield markets on Tempo. Hot-swappable yield strategies. Real-time optimization. You set policy. AI maximizes returns.

Payment streams (salary, contractor payments, vesting schedules) and unvested funds automatically earn yield while waiting to be claimed.
